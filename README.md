# OneSlot – Smart Appointment Booking System

## Project Overview

OneSlot is a web-based appointment booking system developed using PHP and MySQL. It allows users to register, log in, book appointments, manage bookings, and view appointment history. An admin dashboard is also provided to manage bookings.

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP

## Features

* User Registration and Login
* Appointment Booking
* User Dashboard
* Booking Cancellation
* Admin Dashboard
* MySQL Database Integration

## Project Files

* Source Code
* Database Script (database.sql)
* Project Report
* Presentation (PPT)

## How to Run

1. Install XAMPP.
2. Copy project folder to `htdocs`.
3. Create database `oneslot_db`.
4. Import `database.sql` using phpMyAdmin.
5. Open the project in browser:
   `http://localhost/oneslot`
