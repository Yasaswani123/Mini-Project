ONESLOT PHP + MYSQL PROJECT

This is the backend-integrated OneSlot project.

Main files:
- index.html: Existing splash screen landing page
- home.php: Homepage
- register.php: User registration
- login.php: User login
- logout.php: Logout/session destroy
- services.php: Services page
- booking.php: Appointment booking
- dashboard.php: User dashboard
- admin.php: Admin dashboard/login
- config.php: Database and admin settings
- db.php: MySQL connection
- database.sql: Complete database structure
- SETUP_XAMPP.txt: Setup instructions

Splash screen:
The visual splash screen is preserved. The final step redirects to home.php so the website does not stop on a blank screen.

Database:
- users stores registered users
- bookings stores appointments
